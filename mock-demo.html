    怎么做到的呢？

    你只要在.justreq文件添加如下规则，并编写相应的getUsersList.jrs、getUsersInfo.jrs文件即可

    {
  ...
  "rules": [
    {
      "href": "getUsersList.do",
      "subs": "getUsersList.jrs"
    },
    {
      "href": "getUsersInfo.do",
      "subs": "getUsersInfo.jrs"
    }
  ]
}

    本示例所使用jrs文件位于“.jr/subs/jrs/”
